# Финальный проект по Linux/Unix

В этом архиве лежат файлы по моей настройке Fedora VM. Сюда я собрал не только краткое описание, но и выводы команд, чтобы было видно, что именно получилось после настройки.

Что было настроено:

- имя хоста: `mephi-2026.domain.local`
- IPv4-адрес: `192.168.1.100/24`
- шлюз: `192.168.1.1`
- DNS: `8.8.8.8`
- дополнительный диск: `/dev/sdb`
- раздел: `/dev/sdb1`
- файловая система: `ext4`
- метка файловой системы: `MEPHI_DATA`
- точка монтирования: `/data/mephi-web`
- корневая папка nginx: `/data/mephi-web`
- пользователь: `mephi-admin`
- группа: `mephi-devs`
- веб-сервер: `nginx`
- страница: `index.html`
- firewall: `firewalld`, HTTP разрешён

Основные проверки вынесены в отдельные `.txt` файлы. Например, сеть проверялась через `nmcli`, `ip`, `ping` и `curl`, диск — через `lsblk`, `findmnt` и `df`, nginx — через `nginx -t`, `systemctl` и запросы к странице.

Скриншот страницы лежит в файле `mephi-nginx-screenshot.png`.
